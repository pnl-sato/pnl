## 0. このフレームの使い方

1. BizReach 等で見つけた未接触候補者のプロフィールをチャットに貼る
2. 下記5軸（各5点満点・合計25点）で採点する
3. 「評価サマリ（200字以内）」「5軸スコア表」「強調点・懸念点・推奨アクション」「枠判定（メンバー枠／リーダー枠／いずれも不適）」を出力
4. 末尾に「評価履歴.md 追記用1行ブロック」を出力 → コピーしてプロジェクトナレッジの `評価履歴.md` 末尾にペースト

**総合判定の目安**：
- 22〜25点：即推薦準備（スカウト最優先送付）
- 18〜21点：スカウト送付OK、返信あれば優先対応
- 14〜17点：要再評価（追加情報を取って再判定）
- 13点以下：見送り

## 1. 評価軸（5軸 × 5点）

### 軸1：要件適合（5点）

候補者の経験が Must／Want 要件のどこに当てはまるか。**メンバー枠／リーダー枠どちらに該当するかも併記**。

| 点 | 基準 |
| --- | --- |
| 5 | リーダー枠 Must を全て満たし、Want も3つ以上該当（M&A／海外法務／IPO／一人法務／IT・SaaS／資金決済法のいずれか複数） |
| 4 | リーダー枠 Must を満たし、Want も1〜2つ該当。または、メンバー枠で法務経験 3年強 ＋ 主体性高い |
| 3 | メンバー枠 Must を満たす（法務経験〜3年、または法学部卒・キャリアシフト志向） |
| 2 | 法務経験はあるが ToC／決済／IT・SaaS とまったく接点なし。または、メンバー枠でも動機の言語化が弱い |
| 1 | 法務未経験＋ベース知識なし、または専門特化型（知財専業／契約専業のみ） |

### 軸2：カルチャー適合（5点）

「運縁勘」「当事者型」「ガッツ・ストイックさ」「相談しやすさ」とのフィット。

| 点 | 基準 |
| --- | --- |
| 5 | 当事者型志向が明確（事業部と二人三脚の経験／自走経験）＋仕組み化志向＋人柄が柔らかく相談しやすそう |
| 4 | 当事者型志向あり、または一人法務／少人数法務経験で組織化に主体的に巻き込まれた経験あり |
| 3 | スタートアップ／ベンチャー法務経験ありで、整っていない環境への耐性は確認できる |
| 2 | 大手企業法務一筋。整った環境に最適化されている懸念。動機次第で要再判定 |
| 1 | 戦略コンサル／IBD タイトル持ち30〜40代、または「制度が整っているかで選ぶ」志向の明確な発信あり |

### 軸3：年収適合（5点）

実勢オファーゾーン（メンバー枠 〜700万／リーダー枠 800〜900万）との折り合いやすさ。

| 点 | 基準 |
| --- | --- |
| 5 | 現年収が実勢オファーゾーンの中央付近（リーダー枠 800万前後／メンバー枠 600万前後）＋SO理解あり |
| 4 | 現年収が実勢ゾーン下限〜中央。多少のアップ前提で折り合いやすい |
| 3 | 現年収が実勢ゾーン上限ぎりぎり（リーダー枠で 900万弱）。SO・キャリア解放幅で総合納得感の醸成必要 |
| 2 | 現年収 950〜1,050万。「数百万差で折り合わず」リスク高。CFO 藤田氏の年収交渉に強い説得材料が必要 |
| 1 | 現年収 1,100万円超 ＋ 維持希望明示／メンバー枠で 800万円超を希望 |

### 軸4：フェーズ適合（5点）

プライム上場・HD化・M&A・海外展開という事業フェーズへの「キャリア解放幅」としての共鳴度。

| 点 | 基準 |
| --- | --- |
| 5 | HD化・M&A・海外展開フェーズに対し、自分のキャリアの次の解放幅として捉えている発信あり。eギフト・決済領域への能動的興味あり |
| 4 | 急成長フェーズ・上場後の組織拡大局面への適応経験あり。次の挑戦先として違和感なし |
| 3 | スタートアップ→上場局面の経験あり。事業ドメインへの興味は中立 |
| 2 | 安定大手のみの経験。フェーズ理解はあるが自分事化できているか不明 |
| 1 | 「制度が整った安定企業に行きたい」志向／海外移住希望／「2-3年は今の仕事継続」発言あり |

### 軸5：過去パターン適合（5点）

「ギフティで通った／落ちた」パターンとの照合（過去推薦パターン.md 参照）。

| 点 | 基準 |
| --- | --- |
| 5 | 「通る共通点」3つ以上該当（複数業界・複数フェーズ／他社オファー多／横串推進／事業構造深掘り型）。「落ちる共通点」該当ゼロ |
| 4 | 「通る共通点」1〜2つ該当、「落ちる共通点」該当ゼロ |
| 3 | どちらにも強い該当なし。標準ライン |
| 2 | 「落ちる共通点」1つ該当（大手安定企業出身でステップアップ志向弱／制度重視志向／応答が遅い等） |
| 1 | 「落ちる共通点」2つ以上該当、または「No Show・連絡途絶系」の兆候あり |

## 2. 評価サマリの書き方（200字以内）

- 1行目：枠判定（メンバー枠／リーダー枠／いずれも不適）と総合点
- 2行目：候補者の最大の強み（1点）
- 3行目：最大の懸念（1点）
- 4行目：推奨アクション（即推薦／スカウト送付／追加情報取得／見送り）

例：
```
リーダー枠／総合 21/25。SaaS事業会社で一人法務7年、M&A 3件主担当の経験あり、当事者型志向も明確。
懸念は現年収 1,050万でレンジ上限ギリギリ、CFO 藤田氏の年収交渉に説得材料が必要。
推奨：スカウト最優先送付、返信あれば「キャリア解放幅 × SO・HD化フェーズ」で総合納得感の文脈を準備。
```

```markdown

- （3つまで、要件適合・カルチャー適合・過去パターン適合の文脈で）

### 懸念点
- （2つまで、年収／カルチャー／動機の言語化／応答スタイル等）

### 推奨アクション

- 推薦準備：（推薦資料に盛り込むべき材料）
- 追加情報取得：（カジュアル面談で確認すべき項目）
- 見送り：（理由を1行で明示）
```

## 4. 枠判定ガイドライン

### メンバー枠（育成枠）へ振る候補者
- 法務経験〜3年程度、または法学部卒・法務未経験でキャリアシフト志向
- インハウス未経験／弁護士キャリアからのシフト
- 現年収 500〜700万円程度
- ガッツ・ストイックさ・「なぜ企業法務か」の動機が強い

### リーダー枠（経験者）へ振る候補者
- 企業法務 3年以上（マネージャー一歩手前）
- IT／SaaS／インターネットサービス事業会社での経験
- 一人法務／少人数法務での組織化フェーズ経験
- M&A／PMI／海外法務／IPO のいずれか経験
- 現年収 800〜950万円程度
- 仕組み化志向・若手相談役を担える人物性

### いずれも不適
- 法務経験ゼロ＋法学部卒でもない
- 専門特化型（知財専業／契約専業）で3領域横断の主体性が見えない
- 現年収 1,100万円超 ＋ 維持希望
- カルチャー適合 1点該当（戦略コンサル／IBD タイトル持ち、「制度重視」明確発信）

評価のたびに以下のブロックを末尾に出力 → コピーして `評価履歴.md` 末尾にペースト：

```

```

例：
```

```
