# 候補者宛メールエージェント

スカウト返信後〜選考完了までに候補者へ送るメールのガイド。
（スカウト一次接触の文面は `agents/scout.md` を参照。本ファイルは「返信が来た後」「面談後」のメール体）

実サンプルは Gmail 送信箱（2024〜2026 年）から抽出。

---

## 1. シーン一覧

| シーン | 件名パターン | 主な目的 |
|---|---|---|
| 1. 初回コンタクト返信 | `【{媒体}経由でご連絡】Pole&Lineの佐藤です` | スカウト返信に対し日時を確定 |
| 2. 面談前リマインド・遅刻時対応 | （Re: 上記） | 開始直前の確認 |
| 3. 面談後の御礼 | `【面談の御礼】Pole&Lineの佐藤です` | お礼＋次アクション提示 |
| 4. 選考進捗・資料共有 | （継続スレッド） | 進め方の共有、参考資料添付 |
| 5. 書類依頼・確認 | `Pole&Lineの佐藤です` | レジュメや年収情報の確認 |
| 6. 選考結果通知（見送り） | （継続スレッド） | 理由を率直に伝える |
| 7. 過去関係者への再コンタクト | `ご無沙汰しております、Pole&Lineの佐藤です。` | カジュアルな近況伺い |
| 8. 過去関係者への新規案件提案 | `【Pole&Line｜ご挨拶および案件のご提案】{ポジション}` | 案件提案 |
| 9. 個人事情による日程変更 | （継続スレッド） | 率直に事情を伝えて再調整 |

## 2. 面談案内・日程確定

### 書き出し（古市からの引き継ぎ案件）
```
{姓}様
初めまして。私、Pole&Lineの佐藤と申します。
弊社の古市からのメッセージにご返信いただきありがとうございます。
案件にご興味お持ちいただけて嬉しく思います。

古市からもご連絡差し上げているかと思いますが、当日はわたくし佐藤が詳細をご案内差し上げます。
改めて当日詳細を記載しますので、ご確認お願いします。
```

### 日時の書き方（3パターンあり、好みで使い分け）

**パターンA：太罫線**
```
━━━━━━━━━━━━━━━━━━━━━
日時：5月25日（月） 19:00～
URL：https://meet.google.com/xxx-xxxx-xxx
━━━━━━━━━━━━━━━━━━━━━
```

**パターンB：シンプル**
```
---
日時：5月8日（金） 13:00～13:30
URL：https://meet.google.com/tnk-vnkm-txf
---
```

**パターンC：詳細フォーマット（オファー前面談など重要面談）**
```
================================
■オファー前面談

【日時】
11/25 (火) 16:00 - 17:00

【当日の詳細】
・面談担当者：{担当者名}
・所要時間：30分～1時間程度
・場所：オンラインにて実施いたします。

下記URLよりご参加お願いいたします。
================================
```

### 結び
```
ご都合が合わなくなってしまった場合は、お早めにご連絡ください。
よろしくお願いいたします。
```

## 3. 面談前リマインド・遅刻時対応

### 開始直前の確認（候補者が来ない時）

よろしくお願いします。

## 4. 面談後の御礼

### 件名
- `【面談の御礼】Pole&Lineの佐藤です`
- `【ご面談の御礼】Pole&Lineの佐藤です`

### 構造
```
{姓}様
Pole&Lineの佐藤です。

本日はお忙しい中、お時間いただきありがとうございました。
{相手の話で印象に残った点を具体的に1〜2文：「ヘルスケア領域でのご経験やキャリアに対するお考え、大変参考になりました。」「お話を伺いながら、{志向}を次のキャリアで重視されている点が、とても印象に残っています。」}

{次アクションの提示}：
- 紹介する案件のJDや参考情報を添付して送る
- 「お時間あるタイミングで、気になるところだけ拾い読みいただければ大丈夫です。」と負担を軽くする

引き続き{姓}さんのご状況やご志向に合うものがあればご連絡させていただければと思います。
今後もよろしくお願いいたします。

--
Pole&Line合同会社
佐藤 雄太｜SATO Yuta
Mobile：080-3930-1714
```

### ポイント
- 「印象に残った点」は必ず**具体的**に書く（「とても勉強になりました」のような空文句は使わない）
- 候補者の呼び方は本文中は「{姓}さん」もよく使う（御礼メールでの親しみ表現）

## 5. 選考進捗・資料共有

### 候補者へ案件をまとめて打診するとき
```
{姓}様
Pole&Lineの佐藤です。

本日はお忙しい中お時間頂き有難うございました。
{A社}、{B社}につきましてはカジュアル面談から先方に打診させて頂きますね。
{C社}につきましては、上記2社を踏まえた上でまたご相談させてください。

以下{N}社についてまとめた資料ですので、お時間あるときにご一読頂けますと幸いです。

{A社}： {URL}
{B社}： {URL}
{C社}： {URL}

引き続きよろしくお願いします。
```

### JD 共有時のスタイル
- Notion の JD ページを共有：`https://pnl.notion.site/JD-{id}?source=copy_link`
- 自分の言葉でポジションの色合いを2〜3文で添える（例：「各国拠点や現地責任者と連携しながら、全体の状況整理や優先順位付けを行っていく、いわゆる"海外統括"の色合いが強い役割」）

## 6. 書類依頼・確認

### 【マスト】書類依頼前のプライバシーポリシー同意確認

職務経歴書・履歴書など個人情報を含む書類を候補者に依頼する**前**に、SF Contact の **`PersonalInformation1__c`（個人情報の取扱い同意）** を確認する。

| 値 | 対応 |
|---|---|
| `同意書サイン済` または `不要` | 同意リンク添付は**不要**、通常の書類依頼でOK |
| `なし` または空 | **書類依頼メールに同意フォームURLを必ず併記**し、同意 → 書類提出の順で依頼する |

**同意フォームURL：** `https://docs.google.com/forms/d/e/1FAIpQLSf5VhQ8uZ_maUd6wsVy5_RKg6hgLgEE22wiuZvApADb8xOBEw/viewform`
（Craft 正本：「[プライバシーポリシーへの同意](https://docs.craft.do/) `887EB71A-FE4C-4B52-AA03-5BC507EC20F3`」）

同意フォームを送付した候補者は、本人が回答送信した時点で同意完了。回答受領後、SF 側の `PersonalInformation1__c` を `同意書サイン済` に更新する（自動連携あり／手動補正の必要があれば確認）。

#### 書類依頼メールへの組み込み例

```
書類のご共有にあたり、弊社の個人情報の取扱いについてご確認・ご同意のお願いです。
お手数ですが下記フォームにご回答のうえ、書類をお送りいただけますと幸いです。
（所要1〜2分／同意は1回限り、以降の書類依頼では不要となります）

https://docs.google.com/forms/d/e/1FAIpQLSf5VhQ8uZ_maUd6wsVy5_RKg6hgLgEE22wiuZvApADb8xOBEw/viewform
```

Facebook Messenger・LINE 等のチャットの場合も同様に、書類依頼と同じメッセージ内に同意フォームURLを含める。同意がチャネル外（フォーム）で完了するため、書類受領前に必ず分離しておく。

### 確認依頼（複数項目）
```
{姓}様
ご返信有難うございます。
{N}点確認させてください。

【1つ目】
{確認内容1}

【2つ目】
{確認内容2}
```

### 書類体裁の修正依頼
```
一点、書類の体裁についてご相談がございます。
お送りいただいたファイルを弊社にてPDF化した際、{ツール}の仕様により、以下の箇所などで{現象}が発生してしまいました。
※PDF化した職務経歴書は添付しております

・{該当箇所1}
・{該当箇所2}

こちらでも{対応案}できますので、ご都合の良い形でご対応いただけますと幸いです。
```

### 確認OK時
```
内容を確認しました。バッチリです。お手数おかけしました。
こちらで{社名}に打診いたします。打診する際には以前いただいたお話に配慮しながら行います。
引き続きよろしくお願いします。
```

### 個人情報の取扱い確認への応答
```
職務経歴書の取り扱いにつきましても、承知いたしました。
本件は{社名}のポジションに関する検討に限って取り扱い、{姓}様のご承諾なく、他社様へ展開することはございませんのでご安心ください。
```

## 7. 選考結果通知（見送り）

### 構造
1. **結論を最初に明示**
2. **企業側の判断理由を率直に伝える**（候補者を否定しない表現で）
3. **謝罪は短く**
4. **次の話への布石**（あれば）

### 実例パターン1：書類見送り
```
{姓}様
Pole&Lineの佐藤です。

{社名}の件、お待たせしておりました。
ご検討いただいた結果、今回は書類の段階で見送りという判断となりました。

今回のポジションでは{企業が求めていた要素}がより強く期待されていたようで、その点で{姓}様が大切にされている{候補者の志向}とは、若干のズレが{あったようです／生まれてしまったようです}。

良いご縁をお作りできず、申し訳ございません。
{次の話があれば：別件で〜}
```

### 実例パターン2：面談後見送り
```
{社名}の件ですが、面談内容を踏まえて先方に確認したところ、{姓}さんの強みと先方の期待するところにギャップがあるとのご判断でした。
{具体的なギャップ}：「○○のご経験は魅力に映る一方で、△△における経験の厚みとビジネスモデルの違いが課題とのことです。」

良いご縁をお作りできず、申し訳ございません。
別件{で〜／また機会あればご相談させてください}
```

### ポイント
- 企業側の言葉として伝える（「先方のご判断」）→ エージェントの主観ではない
- ネガティブを和らげる：「ズレが生まれてしまったようです」「ギャップがある」
- 「良いご縁をお作りできず、申し訳ございません」は定番フレーズ

## 8. 過去関係者への再コンタクト

### カジュアル再開（よく知っている候補者）
```
{姓}様
ご無沙汰してます、Pole&Lineの佐藤です。

{過去のやり取りに触れる}：去年は転職の件でいろいろお時間いただき、ありがとうございました。その後ご連絡できておらず失礼しました。

{再連絡のきっかけ}：先日、{知人}とランチした際に、{近況情報}と伺い、ふと思い出してご連絡しました。

その後いかがですか？
もしよければ近況含めてキャッチアップできればと思っていて、今度お茶かランチでもご一緒できればと考えていますが、ご都合いかがでしょうか？
```

### フォーマル再開（疎遠な過去関係者へ案件提案）
件名：`【Pole&Line｜ご挨拶および案件のご提案】{ポジション要素}`
```
{姓}様
突然のご連絡失礼致します。
私、サーチ型人材エージェント会社のPole&Line 佐藤と申します。

{姓}様とは以前（{年月}ごろ）、弊社の{担当者}というコンサルタントが{媒体}経由でコンタクト、やり取りさせて頂いておりました。

現在、{ポジションの概要}を担っていただける方を探しており、{姓}様のご経歴を拝見し、ご連絡させていただきました。

{ポジション詳細を簡潔に}

本件、一般的な〜
```

### LinkedIn等での近況に触れた再アプローチ
```
{姓}様
ご無沙汰しております。Pole&Lineの佐藤です。

先日、{社名}でのインタビュー記事を拝見し、現在{担当領域}を管掌されているとのことで、ご活躍の様子を興味深く拝見しておりました。

少しご相談したいテーマがありご連絡しました。
{案件概要}
```

## 9. 個人事情による日程変更

率直に事情を書く。長々と謝らない。

```
{姓}様
Pole&Lineの佐藤です。

先日はお時間頂きありがとうございました。
{予定}のお時間頂戴しておりましたが、こちら延期させていただけないでしょうか。{事情を1文で：昨日、子供が感染性胃腸炎を発症し、本日わたしまで感染してしまったようで…}申し訳ございません。
```

## 11. 共通ルール

- 件名は `Pole&Line` を含める（社内識別のため）
- 候補者宛の本文中での呼び方は **「{姓}様」**（堅め）または **「{姓}さん」**（親密な関係・面談後）を使い分け
- 署名は基本フル版：
  ```
  --
  Pole&Line合同会社
  佐藤 雄太｜SATO Yuta
  Mobile：080-3930-1714
  Mail：sato-y@pnl.co.jp
  Website：https://pnl.co.jp
  ```
- 古市と佐藤の役割分担を明示する（「弊社の古市から〜」「当日はわたくし佐藤が〜」）
- 「お忙しい中」「お時間頂き」など相手への配慮表現を入れる
- 結果通知や調整失敗は**短く謝り、原因を率直に伝える**。長文の言い訳はしない。
