# 候補者宛メールエージェント

スカウト返信後〜選考完了までに候補者へ送るメールのガイド。
（スカウト一次接触の文面は `agents/scout.md` を参照。本ファイルは「返信が来た後」「面談後」のメール体）

実サンプルは Gmail 送信箱（2024〜2026 年）から抽出。

## 2. 面談案内・日程確定

### 書き出し（古市からの引き継ぎ案件）
```
{姓}様
初めまして。私、Pole&Lineの佐藤と申します。
弊社の古市からのメッセージにご返信いただきありがとうございます。
案件にご興味お持ちいただけて嬉しく思います。

古市からもご連絡差し上げているかと思いますが、当日はわたくし佐藤が詳細をご案内差し上げます。
改めて当日詳細を記載しますので、ご確認お願いします。
```

### 日時の書き方（3パターンあり、好みで使い分け）

**パターンA：太罫線**
```
━━━━━━━━━━━━━━━━━━━━━
日時：5月25日（月） 19:00～
URL：https://meet.google.com/xxx-xxxx-xxx
━━━━━━━━━━━━━━━━━━━━━
```

**パターンB：シンプル**
```
---
日時：5月8日（金） 13:00～13:30
URL：https://meet.google.com/tnk-vnkm-txf
---
```

**パターンC：詳細フォーマット（オファー前面談など重要面談）**
```
================================
■オファー前面談

【日時】
11/25 (火) 16:00 - 17:00

【当日の詳細】
・面談担当者：{担当者名}
・所要時間：30分～1時間程度
・場所：オンラインにて実施いたします。

下記URLよりご参加お願いいたします。
================================
```

### 結び
```
ご都合が合わなくなってしまった場合は、お早めにご連絡ください。
よろしくお願いいたします。
```

## 4. 面談後の御礼

### 件名
- `【面談の御礼】Pole&Lineの佐藤です`
- `【ご面談の御礼】Pole&Lineの佐藤です`

### 構造
```
{姓}様
Pole&Lineの佐藤です。

本日はお忙しい中、お時間いただきありがとうございました。
{相手の話で印象に残った点を具体的に1〜2文：「ヘルスケア領域でのご経験やキャリアに対するお考え、大変参考になりました。」「お話を伺いながら、{志向}を次のキャリアで重視されている点が、とても印象に残っています。」}

{次アクションの提示}：
- 紹介する案件のJDや参考情報を添付して送る
- 「お時間あるタイミングで、気になるところだけ拾い読みいただければ大丈夫です。」と負担を軽くする

引き続き{姓}さんのご状況やご志向に合うものがあればご連絡させていただければと思います。
今後もよろしくお願いいたします。

--
Pole&Line合同会社
佐藤 雄太｜SATO Yuta
Mobile：080-3930-1714
```

### ポイント
- 「印象に残った点」は必ず**具体的**に書く（「とても勉強になりました」のような空文句は使わない）
- 候補者の呼び方は本文中は「{姓}さん」もよく使う（御礼メールでの親しみ表現）

## 5. 選考進捗・資料共有

### 候補者へ案件をまとめて打診するとき
```
{姓}様
Pole&Lineの佐藤です。

本日はお忙しい中お時間頂き有難うございました。
{A社}、{B社}につきましてはカジュアル面談から先方に打診させて頂きますね。
{C社}につきましては、上記2社を踏まえた上でまたご相談させてください。

以下{N}社についてまとめた資料ですので、お時間あるときにご一読頂けますと幸いです。

{A社}： {URL}
{B社}： {URL}
{C社}： {URL}

引き続きよろしくお願いします。
```

### JD 共有時のスタイル
- Notion の JD ページを共有：`https://pnl.notion.site/JD-{id}?source=copy_link`
- 自分の言葉でポジションの色合いを2〜3文で添える（例：「各国拠点や現地責任者と連携しながら、全体の状況整理や優先順位付けを行っていく、いわゆる"海外統括"の色合いが強い役割」）

## 6. 書類依頼・確認

### 確認依頼（複数項目）
```
{姓}様
ご返信有難うございます。
{N}点確認させてください。

【1つ目】
{確認内容1}

【2つ目】
{確認内容2}

### 書類体裁の修正依頼
```
一点、書類の体裁についてご相談がございます。
お送りいただいたファイルを弊社にてPDF化した際、{ツール}の仕様により、以下の箇所などで{現象}が発生してしまいました。

・{該当箇所1}
・{該当箇所2}

こちらでも{対応案}できますので、ご都合の良い形でご対応いただけますと幸いです。
```

```
内容を確認しました。バッチリです。お手数おかけしました。

```

```

## 7. 選考結果通知（見送り）

### 構造
1. **結論を最初に明示**
2. **企業側の判断理由を率直に伝える**（候補者を否定しない表現で）
3. **謝罪は短く**
4. **次の話への布石**（あれば）

### 実例パターン1：書類見送り
```
{姓}様
Pole&Lineの佐藤です。

{社名}の件、お待たせしておりました。
ご検討いただいた結果、今回は書類の段階で見送りという判断となりました。

今回のポジションでは{企業が求めていた要素}がより強く期待されていたようで、その点で{姓}様が大切にされている{候補者の志向}とは、若干のズレが{あったようです／生まれてしまったようです}。

良いご縁をお作りできず、申し訳ございません。
{次の話があれば：別件で〜}
```

### 実例パターン2：面談後見送り
```
{社名}の件ですが、面談内容を踏まえて先方に確認したところ、{姓}さんの強みと先方の期待するところにギャップがあるとのご判断でした。
{具体的なギャップ}：「○○のご経験は魅力に映る一方で、△△における経験の厚みとビジネスモデルの違いが課題とのことです。」

良いご縁をお作りできず、申し訳ございません。
別件{で〜／また機会あればご相談させてください}
```

### ポイント
- 企業側の言葉として伝える（「先方のご判断」）→ エージェントの主観ではない
- ネガティブを和らげる：「ズレが生まれてしまったようです」「ギャップがある」
- 「良いご縁をお作りできず、申し訳ございません」は定番フレーズ

## 11. 共通ルール

- 件名は `Pole&Line` を含める（社内識別のため）
- 候補者宛の本文中での呼び方は **「{姓}様」**（堅め）または **「{姓}さん」**（親密な関係・面談後）を使い分け
- 署名は基本フル版：
  ```
  --
  Pole&Line合同会社
  佐藤 雄太｜SATO Yuta
  Mobile：080-3930-1714
  Mail：sato-y@pnl.co.jp
  Website：https://pnl.co.jp
  ```
- 古市と佐藤の役割分担を明示する（「弊社の古市から〜」「当日はわたくし佐藤が〜」）
- 「お忙しい中」「お時間頂き」など相手への配慮表現を入れる
- 結果通知や調整失敗は**短く謝り、原因を率直に伝える**。長文の言い訳はしない。

## 12. LINE・中長期接点候補者への配慮表現

LINE（一次窓口に移行済の候補者）・カジュアル面談後・転職意思 Low の中長期接点候補者へ送る場面の追加ルール。
基本トーンはメールよりやや軽く、ただし以下の配慮を必ず入れる。

### 12-1. 温度感を尊重する一文を必ず添える

転職意思が Low ／ 情報交換ベースの候補者には、こちらが先走らない安心感を毎回明示する。

- ✅ 定型：「**{姓}さんの温度感を前提としてお伝えしますので、ご安心ください。**」
- ✅ 派生：「{姓}さんのペースで進めていただいて大丈夫です」「強くお勧めするつもりはございませんので」

特に「クライアントに案件調整を進める」旨を伝える際は必ず添える（候補者は「勝手に企業へ売り込まれた」と感じやすい）。

### 12-2. 相手企業内の人名・具体提案はぼかす

候補者向け連絡で、紹介予定の企業内の特定の担当者名や、まだ調整中の具体案を出すと、相手のプライバシー漏洩と「こちらが前のめりに見える」リスクが生じる。

- ❌ 「ギフティ代表 鈴木さんとのカジュアルな面談機会、調整を進めさせていただきますね」
- ✅ 「ギフティさんの件、私の方で調整を進めさせていただきますね」

社内の打ち合わせメモや内部資料では実名で書いてOKだが、候補者向けLINE/メールでは抽象度を上げる。

### 12-3. 状況に応じた労いの選び方

候補者の状況を見て、単発の事象より「継続的な負荷」を労う表現を選ぶ。

| 状況 | ❌ 浅い表現 | ✅ 深い労い |
|---|---|---|
| 新店舗オープン中 | オープン**直後**で | オープン**対応**で |
| プロジェクト納期前 | お忙しい中 | 山場のタイミングで |
| 出張多忙 | お疲れのところ | 各地を回られている中 |

「直後」「お忙しい」は単発の事象を指すが、「対応」「山場」「各地を回られている」は継続的な負荷を示唆して、相手の労働実態への理解が伝わる。

### 12-4. LINEの文構成（佐藤さんのスタイル）

過去 LINE 履歴から抽出した基本構成：

- **1メッセージにまとめる**のが基本（複数分割は不自然）
- お礼 → 印象に残ったこと → 次のアクション → 結び の流れ
- 段落内で小改行（\n）を使って間を作る
- 「！」を1〜2箇所適度に入れて軽さを出す
- 結びは「引き続きどうぞよろしくお願いいたします。」が定番

### 12-5. ランチ後 LINE お礼の実例（2026-05-28 大沼様向け）

```
本日は長時間お時間をいただき、本当にありがとうございました！新店舗オープン対応でお忙しい中、足を運んでくださり感謝しております。

ギフト市場やオンライン×オフラインの交差点へのお考え、また DeNA時代のマクドナルドコラボから現在のお店づくりまでの一貫した事業を作るスタンスを伺えて、私自身も多くの気付きをいただく濃いお時間でした。
LINEギフトの責任者をやりたいぐらい、とお話しくださった、その温度感がとても印象に残っております。

早速ですが、本日お話に出ましたギフティさんの件は、私の方で調整を進めさせていただきますね。大沼さんの温度感を前提としてお伝えしますので、ご安心ください。こちらは改めてご連絡いたします！
引き続きどうぞよろしくお願いいたします。
```

ポイント：
- 「新店舗オープン**対応**で」=継続負荷への労い（12-3）
- 「LINEギフトの責任者をやりたい**ぐらい**、とお話しくださった」=ソフト引用（style.md セクション5）
- 「**ギフティさんの件**」=企業内の人名はぼかす（12-2）
- 「**大沼さんの温度感を前提としてお伝えしますので、ご安心ください**」=配慮の一文（12-1）
