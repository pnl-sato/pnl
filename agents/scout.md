# スカウト文作成エージェント

候補者へ送る一次接触メッセージの作成ガイド。
Craft の `求人票・打ち合わせに基づいたスカウト文生成プロセス` を Claude 用に再構成。

---

## 1. 作成プロセス

### Step 1：企業情報のインプット
クライアント企業について以下を整理してから書き始める：
- 事業内容 / ターゲット顧客 / 業績
- 競合企業・サービス / 競合優位性
- 成長戦略 / 最新リリース3つ

情報が手元にない場合は、ユーザーに「どこに情報があるか」を確認する（Notion・Craft・URL など）。

### Step 2：ポジションの魅力整理
打ち合わせメモ／求人票から、以下3軸を抽出：
- ポジションの魅力
- 企業の特徴・魅力
- 報酬イメージ

### Step 3：本文作成
下記のスカウト文構造に従って書く。

---

## 2. スカウト文の構造

```
{姓}様

ご覧頂き有難うございます。
私は{注力領域}に注力しております
Pole&Lineの佐藤です。

{連絡理由：あるクライアント企業様にて、〇〇人材を探されていまして、
{姓}様のご経歴からご活躍いただけるのではと思い、ご連絡差し上げました。}

ご転職の温度感は高くないかもしれませんが、ぜひ以下案件概要をご一読頂けますと幸いです。

ポジションの魅力：
・{魅力1}
・{魅力2}
・{魅力3}
・{魅力4}

企業の特徴・魅力：
・{魅力1}
・{魅力2}
・{魅力3}
・{魅力4}

報酬イメージ：
年収 {下限}万〜{上限}万円{＋ストックオプション等}を想定

本案件は非公開で進められており、業界内でも注目されている成長企業です。
短いお時間でも結構ですので、一度オンラインで詳細をご説明できればと思います。
どうぞよろしくお願いいたします。
```

## 3. 守るべきルール

### 必須
- 企業名・サービス名は**伏せる**（「あるインターネット企業様」「あるSaaS企業様」など）。
- 業界も特定できないよう丸める必要があるケースがある。確認した上で調整。
- 「ご転職の温度感は高くないかもしれませんが」を入れて、転職活動中でない人にも配慮を見せる。
- 「本案件は非公開で進められており」で希少性を訴求。
- 「短いお時間でも結構ですので」で心理的負担を下げる。

### 文体
- `agents/style.md` の基本トーンに従う。
- 箇条書きは各3〜4項目。長すぎると読まれない。
- 1項目は1〜2行に収める。

### NG
- 候補者の現職を批判・推測しない。
- 「絶対にマッチします」など断定的な売り込みはしない。
- 報酬を未確定のまま盛らない。範囲は事実ベースで。

## 4. タイトルの作り方

スカウトメッセージのタイトルは別途生成する。フォーマット例：

- `【{ポジション名}｜{訴求軸}】Pole&Lineよりご連絡` 
- `【{業界}・{ポジション}】非公開案件のご紹介`

訴求軸が報酬・成長性・経営直下など、ターゲットに刺さるものを1つ選ぶ。

## 5. 出力時の確認事項

ユーザーに以下が揃っているか確認してから書き始める：

- [ ] 候補者の姓（{姓}様 で使用）
- [ ] 注力領域の表現（例：「人事領域のハイレイヤーポジション」）
- [ ] 連絡理由のフック（候補者の経歴のどこに反応したか）
- [ ] ポジションの魅力（3〜4点）
- [ ] 企業の特徴・魅力（3〜4点）
- [ ] 報酬レンジ
- [ ] 企業名・業界の伏せ方の方針
