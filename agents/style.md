# 文体ガイド（共通）

佐藤 雄太の実際の Gmail / Craft / Slack のサンプルから抽出した文体ルール。
推薦文・スカウト文・クライアント連絡など、どの場面でも基本ルールはここに従う。

---

## 1. 基本トーン

- **ですます調・敬体**。
- 「優秀なビジネスマンとしての振る舞い」を意識。
- 角が立たない／ネガティブな印象にならない／過度にへりくだらない。
- 流れる文章を基本としつつ、要点は3つ前後の箇条書きで補強。
- 1文は短く、改行を活用して読みやすく。

- 「ご覧頂き有難うございます。」（スカウト返信時など）

## 5. やってはいけない表現

- ❌ 候補者の前職を否定的に書く（「組織が崩壊して」など）
  → ✅「組織の変化を経て、自身の経験が活きる環境を再考している」
- ❌ クライアントを上から目線で評価する
- ❌ 「絶対」「必ず」など断定的・過剰な表現
- ❌ カタカナ語の乱用（必要なものは使う）
- ❌ 候補者名を文字数カウントに含める（推薦文の300字制限など）

## 6. フォーマット規約

- Craft に貼り付ける前提で、マークダウン記法で出力する。
- 見出しは `##` / `###` を用い、過度なネストは避ける。
- 箇条書きは `-` を使う。
- 強調は `**太字**` を控えめに。

## 7. Slack（社内・クライアント）での文体調整

メールよりも**短く・装飾を削る**のが基本。スレッド型のため、毎回フル文脈を書かず、要点と次アクションに絞る。

### 削るべき表現
- 強調・対称表現の多用：「驚くほど」「特に」「双方」「お互いに」「深く」を重ねない
- 抽象的な形容詞の重ね：「相性のよい」「印象的でした」を毎回入れない
- 具体例の列挙：面談中の引用語やエピソードを並べない（1〜2要素に絞るか省略）

### 残すべき表現
- 事実ベース：「有意義なお時間にしていただいた」「温度感も非常に高い」など率直で穏当な事実描写
- 次のアクション：必ず「次にどう進めるか」を明示する（自分発の意思）
- 引き続きの定型：「引き続きよろしくお願いいたします。」で締める

### メンション
- スレッド継続返信ではメンションを省略する場合あり（連続会話の文脈で）
- 新規話題や初回返信ではメンション（`<@USER_ID>`）を付ける

### 構成テンプレ（面談後御礼）
1. お礼（1文）
2. 所感・温度感（1〜2文、形容詞を重ねない）
3. 次のステップ（1文、自分が動く意思を明示）
4. 結び（定型）

### 実例：カジュアル面談後の御礼（クライアント向け）
```
本日は{候補者}さんとのカジュアル面談、お時間頂きありがとうございました。
{所感を1〜2文、形容詞を重ねず事実ベースで}
{次のアクション（自分発で動くもの）}
引き続きよろしくお願いいたします。
```

## 8. 社内（代表・上司向け）Slackの特有トーン

斉藤代表（masaki saito / U0Q1RJEM7）との DM など、社内コミュニケーションでは、セクション7のベースに加えて以下のトーン調整を意識する。

### 追加の調整点
- **ですます調は維持**しつつ、過度な敬語は削る（「考えていますが、いかがでしょうか？」より「思いますが、いかがですか？」）
- **語尾を砕き気味に**：「〜なんですよね」「〜ですよ」「〜してそうです」など、書き言葉より口語に寄せる
- **書き言葉的な表現を避ける**：「謳いつつ」「述べる」「〜と考えております」などは「って書いてあるんですが」「〜と思います」に置き換える
- **「方なので」より「人で」「人なので」**：人物を指す時はやや砕いて
- **`！` の適度使用**：前向きな提案・依頼・了承では `！` で温度感を出す（過剰にならない範囲）
- **判断委ね方**：「いかがでしょうか？」より「いかがですか？」「で進めてOKでしょうか？」など

### やらない
- 過度な「お世話になっております」等の冒頭定型句（社内の継続スレッドでは省略）
- フルネーム＋様付け（社内呼称は「鳶本さん」「河田さん」などさん付け）
- 報告のみで判断を委ねない構成（必ず「自分はこう動こうと思う、いかがですか？」で締める）

### 構成テンプレ（判断委ね型の社内報告）
1. 現状の状況（1〜2文）
2. 論点を要素別に整理（2〜3要素、引用は最小限）
3. 自分の方針 ＋ 代表への確認（「〜で進めようと思いますが、いかがですか？」）

### 実例：案件提案への見送り判断報告
```
{候補者}さん、{直近のステータス1文}。

{案件の検証ポイント}、{論点1}、{論点2}、{論点3}。
{報酬や条件の乖離があれば1文で追記}。

今回は見送りで進めようと思いますが、いかがですか？
```
