# 文体ガイド（共通）

佐藤 雄太の実際の Gmail / Craft / Slack のサンプルから抽出した文体ルール。
推薦文・スカウト文・クライアント連絡など、どの場面でも基本ルールはここに従う。

---

## 1. 基本トーン

- **ですます調・敬体**。
- 「優秀なビジネスマンとしての振る舞い」を意識。
- 角が立たない／ネガティブな印象にならない／過度にへりくだらない。
- 流れる文章を基本としつつ、要点は3つ前後の箇条書きで補強。
- 1文は短く、改行を活用して読みやすく。

## 2. 敬称・呼称

| 対象 | 呼称 |
|---|---|
| 候補者（本人宛） | `{姓}様` |
| 候補者（第三者への言及） | `{姓}様` / `{姓 名}様` |
| クライアント企業（宛先） | `貴社` / `〇〇株式会社御中` |
| クライアント担当者 | `{姓}様` |
| 自社名 | `Pole&Line` / `Pole&Line合同会社` |
| 自身 | `Pole&Lineの佐藤` / `佐藤 雄太` |

## 3. 頻出フレーズ

### 書き出し
- 「お世話になっております。Pole&Lineの佐藤です。」
- 「はじめまして。Pole&Lineの佐藤と申します。」
- 「ご覧頂き有難うございます。」（スカウト返信時など）

### 連絡理由
- 「〜について、要件の確認でご連絡いたしました。」
- 「ご経歴を拝見し、ご活躍いただけるのではと思い、ご連絡差し上げました。」
- 「ぜひ一度お打ち合わせのお時間を頂戴したくご連絡しました。」

### 提案・推し
- 「〜という印象を持っております。」
- 「〜とお見受けしており、貴社にマッチするのではと考えております。」
- 「〜にご関心をお持ちです。」
- 「短いお時間でも結構ですので、一度オンラインで詳細をご説明できればと思います。」

### 結び
- 「どうぞよろしくお願いいたします。」
- 「引き続きよろしくお願いいたします。」
- 「ご検討いただけますと幸いです。」

### 候補者の転職理由を伝えるとき（必ずポジティブに）
- 「現職への不満はなく、〜のフェーズを志向。」
- 「より責任範囲が大きいところに挑戦したい」
- 「次のステージに向けて、現職の中ではなく外部で挑戦したい」

## 5. やってはいけない表現

- ❌ 候補者の前職を否定的に書く（「組織が崩壊して」など）
  → ✅「組織の変化を経て、自身の経験が活きる環境を再考している」
- ❌ クライアントを上から目線で評価する
- ❌ 「絶対」「必ず」など断定的・過剰な表現
- ❌ カタカナ語の乱用（必要なものは使う）
- ❌ 候補者名を文字数カウントに含める（推薦文の300字制限など）
- ❌ 「〜という整理でした」（やや事務的・分析的すぎて佐藤の語り口に合わない）
  → ✅「〜のようです」「〜とのことでした」など、先方の判断としてやわらかく伝える

## 6. フォーマット規約

- Craft に貼り付ける前提で、マークダウン記法で出力する。
- 見出しは `##` / `###` を用い、過度なネストは避ける。
- 箇条書きは `-` を使う。
- 強調は `**太字**` を控えめに。
