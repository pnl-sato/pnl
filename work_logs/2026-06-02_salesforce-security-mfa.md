## 実施した対応

- **佐藤の管理者アカウントに「組み込み Authenticator（MacBook Air の Touch ID）」を登録完了**
  - 手順：設定 → 私の個人情報 → 高度なユーザーの詳細 → ページ最下部「組み込み Authenticator」→「追加」
  - Chrome のパスキー保存先で **iCloud キーチェーン**を選択（Mac は Touch ID / iPhone は Face ID で利用可、iCloud 同期＆バックアップされる）
  - 既存の Salesforce Authenticator アプリは予備として保持
  - **Touch ID での再ログイン成功を確認**（パスワード＋指のせで完了、スマホ承認より省手数）
- 組織側の許可設定（組み込み認証器・セキュリティキー）は既に ON のため追加変更なし
